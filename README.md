SkySense 🌦️
A smart Android weather app with real-time forecasts and AI-powered daily weather recommendations built with Kotlin and Jetpack Compose and integrated with weather API.

Features
Real-time weather information
City-based weather search
Temperature, humidity, and wind details
Weather forecast
Smart Weather Assistant
Clothing recommendations
Hydration suggestions
Daily weather advice
Technologies Used
Kotlin
Jetpack Compose
MVVM Architecture
Retrofit
Open-Meteo Weather API
Purpose
Developed as a mobile computing project demonstrating API integration and modern Android development.
